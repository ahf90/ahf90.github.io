My Work

This album contains most of my web-development and statistics-related work.

FantasyRig

FantasyRig is a Chrome extension that allows you to view Sabermetrics for your players in ESPN and Yahoo fantasy baseball.

jQuery, Chrome extension, python, fantasy baseball, Sabermetrics

NBA Cluster Analysis

Uses k-means clustering to cluster NBA players based on user-selectable statistics. Returns cluster assignments, cluster centroids, and "Defense vs. Cluster" for each NBA team.

python, google app engine, jQuery, cluster analysis, k-means clustering, statistics, NBA, DFS

NBA Correlations

Compares and calculates correlations for NBA player's fantasy sports scores.

python, google app engine, jQuery, statistics, NBA, DFS

NBA Projections

Daily fantasy sports projections for NBA players.

python, google app engine, Google Apps Script, projections, k-means clustering, cluster analysis, numpy, statistics, NBA, DFS

NCAAB Projections

Daily fantasy sports projections for college Basketball. The website displayed editable projections and a lineup optimizer. The site is deprecated because college daily fantasy sports no longer exists. A demo remains to show how the website worked.

python, google app engine, AngularJS, Chrome extension, knapsack algorithm, projections, k-means clustering, cluster analysis, statistics, numpy, Vegas, CBB, DFS, etc.

HooScript
